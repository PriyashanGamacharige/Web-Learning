Courses Offered


Web Development

Web Development focuses on building websites and web applications. Students learn how to create both the front-end (what users see) and back-end (server-side) parts of a website. The course covers HTML, CSS, JavaScript, and server-side technologies.

Course Content:

  1. HTML & CSS Basics
  2. Responsive Web Design
  3. JavaScript Fundamentals
  4. Backend Development (Node.js, PHP)
  5. Databases and SQL


Creative Design

Creative Design teaches how to design visual content for websites, apps, and marketing materials. The course emphasizes the principles of design, such as color theory, typography, and layout, and how to use design tools to create visually appealing designs.

Course Content:

  1. Design Principles
  2. Color Theory
  3. Typography in Design
  4. Using Adobe Photoshop and Illustrator
  5. Creating Visual Content for Web & Print


UX/UI Design

UX/UI Design focuses on creating websites and applications that are easy to use and visually attractive. Students learn how to design user experiences (UX) and user interfaces (UI) that improve how users interact with digital products.

Course Content:

  1. User Experience (UX) Fundamentals
  2. User Interface (UI) Design Principles
  3. Wireframing and Prototyping
  4. Usability Testing
  5. Design Tools (Sketch, Figma)


Web Content Management

Web Content Management teaches how to organize and manage the content of a website. Students learn how to use Content Management Systems (CMS) to update and maintain websites easily without technical knowledge.

Course Content:

  1. Introduction to CMS (WordPress, Joomla)
  2. Content Creation and Editing
  3. SEO Best Practices
  4. Managing Media and Files
  5. Site Performance Optimization


Web Programming

Web Programming focuses on the coding aspect of web development. Students learn programming languages used to create dynamic websites and applications, such as JavaScript, Python, and PHP.

Course Content:

  1. Introduction to JavaScript
  2. Python for Web Development
  3. PHP Programming Basics
  4. Building Dynamic Web Pages
  5. APIs and Web Services
